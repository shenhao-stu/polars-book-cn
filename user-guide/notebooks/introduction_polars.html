Polars - User Guide

import polars as pl

表达式

fn(Series) -> Series

  • 懒惰评估
    • 可以被优化
    • 向库提供背景信息，并做出明智的决定
  • 高度并行
  • 上下文相关
    • 选择 / 投影 -> Series = 列、文本或值
    • 聚合 -> Series = 组
df = pl.DataFrame(
    {
        "A": [1, 2, 3, 4, 5],
        "fruits": ["banana", "banana", "apple", "apple", "banana"],
        "B": [5, 4, 3, 2, 1],
        "cars": ["beetle", "audi", "beetle", "beetle", "beetle"],
        "optional": [28, 300, None, 2, -30],
    }
)

(df.select([
    pl.col("A"),
    "B",      # "B"这一列将会被推理
